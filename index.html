Usage
This tool helps to distribute Ethereum to a large amount of addresses. Fill in the fields: address (should start with 0x), ETH amount, gas price (gwei), your private key (will only be used to send Ethereum, we DO NOT HAVE ACCESS TO YOUR KEY). Nonce will be calculated automatically. Check the checkbox and press 'fire!'. Wait for clock icons to get replaced with transactions' status. You can hover over a cross to see why the transaction has failed, or click a tick to see the transaction on Etherscan. Note that Etherscan sees if a transaction is confirmed a bit later, so don't get scared if you see a green tick and 'Pending' on Etherscan.
In some (very rare) cases a transaction may get confirmed, but then fall out of the main chain if a fork occurs, so a tick means that a transaction will almost definitely stay in the chain.
